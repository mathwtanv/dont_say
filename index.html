To Say or Not to Say a Number

Game Rules

Welcome to To Say or Not to Say. The default version of this game is 'Don't Say 13', which you can go to by directly pressing the Start Game button []. This is a two player game, and the aim is to either not land on (Don't Say mode) or land on (Say mode) the number specified in the second dropdown. Once this number is hit, the game ends. On each turn, you can add on to the number your opponent leaves you with, by pressing one of the say buttons []. The largest value is what is being referred to as the increment.
